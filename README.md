# パスワードジェネレーター


## 使い方
１：「文字列」からパスワードに含める文字列を選択する。

２：「文字数」から文字数を選択する。

３：「個数」から生成するパスワードの数を選択する。

４：「生成」ボタンを押下すると、黒枠内にパスワードが表示される。

５：パスワードをクリックするとコピーされる。
